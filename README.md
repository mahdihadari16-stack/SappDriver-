## 📱 درباره پروژه

این اپلیکیشن یک نسخه محلی از اپلیکیشن تاکسی آنلاین برای راننده‌های شهر **بندرعباس** و استان **هرمزگان** است. راننده‌ها می‌توانند از طریق این اپلیکیشن درخواست‌های سفر را دریافت کرده و خدمات حمل و نقل ارائه دهند.

## ✨ امکانات

### 🏠 **صفحه اصلی**
- نمایش وضعیت آنلاین/آفلاین راننده
- آمار روزانه (تعداد سفرها و درآمد)
- دسترسی سریع به عملکردهای اصلی
- وضعیت فعلی و منطقه فعالیت

### 🗺️ **نقشه تعاملی**
- نمایش نقشه دقیق شهر بندرعباس
- نشانگرهای مکان‌های مهم:
  - مرکز شهر بندرعباس
  - فرودگاه بندرعباس
  - بندر شهید رجایی
- امکان مسیریابی و تعیین موقعیت فعلی

### 📋 **مدیریت درخواست‌ها**
- نمایش درخواست‌های سفر در زمان واقعی
- اطلاعات کامل هر سفر (مبدأ، مقصد، مسافت، قیمت)
- امکان پذیرش یا رد درخواست‌ها

### ⚙️ **عملکردهای جانبی**
- مدیریت پروفایل راننده
- مشاهده سابقه سفرها
- گزارش‌گیری درآمد
- تنظیمات اپلیکیشن

## 🛠️ نصب و راه‌اندازی

### پیش‌نیازها

```bash
Python 3.8+
pip (Python package manager)
```

### مرحله ۱: نصب وابستگی‌ها

```bash
# نصب KivyMD و Kivy
pip install kivymd kivy

# نصب کتابخانه نقشه
pip install kivy-garden.mapview
```

### مرحله ۲: کلون کردن پروژه

```bash
git clone https://github.com/yourusername/snapp-driver-bandarabbas.git
cd snapp-driver-bandarabbas
```

### مرحله ۳: اجرای اپلیکیشن

```bash
python main.py
```

## 🚀 نحوه استفاده

### برای راننده‌ها:

1. **ثبت‌نام**: ابتدا پروفایل خود را کامل کنید
2. **آنلاین شدن**: روی دکمه "شروع کار" کلیک کنید
3. **دریافت درخواست**: منتظر درخواست‌های سفر باشید
4. **پذیرش سفر**: درخواست مورد نظر را پذیرفته و به مقصد بروید
5. **اتمام سفر**: پس از رسیدن، سفر را به پایان برسانید

### مناطق پوشش:

- **شهر بندرعباس** (مرکز استان)
- **بندر شهید رجایی**
- **فرودگاه بین‌المللی بندرعباس**
- **دانشگاه هرمزگان**
- **مناطق صنعتی هرمزگان**

## 🌐 مختصات جغرافیایی

# مرکز بندرعباس
BANDAR_ABBAS_CENTER = {
